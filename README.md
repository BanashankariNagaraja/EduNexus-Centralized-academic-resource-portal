# EduNexus – Centralized Academic Resource Portal

## Overview
EduNexus is a Flask-based web application designed to provide a centralized platform for managing and accessing academic resources. Students can securely access semester-wise study materials, notes, and books, while administrators can manage users, courses, subjects, and uploaded resources.

## Features
- User Registration and Login
- Role-Based Authentication (Admin & Student)
- Semester-wise Study Materials
- PDF Notes and Book Management
- Course and Subject Management
- Feedback System
- Responsive User Interface

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap
- JavaScript

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/edunexus.git
   ```
2. Navigate to the project folder:
   ```bash
   cd edunexus
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python app.py
   ```
